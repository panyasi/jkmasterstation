产品经理

即将发布，敬请期待

产品经理是这几年最火热的互联网职位之一，基本上是各个互联网公司的标准配置。负责调查并根据用户的需求，确定开发何种产品，选择何种商业模式等等。一个成功的产品经理不但能引导产品的发展，而且能引导公司的发展。

UED设计

交互设计师

即将发布，敬请期待

交互设计师（InteractionDesign）是指设计师对产品与它的使用者之间的互动机制进行分析、预测、定义、规划、描述和探索的过程，即定义人造物的行为方式。交互指的是产品与它的使用者之间的互动过程，而交互设计师则是秉承以用户为中心的设计理念，以用户体验度为原则，对交互过程进行研究并开展设计的工作人员。
UI设计师

即将发布，敬请期待

“UI”的本义是用户界面，是英文User和interface的缩写。UI设计师简称UID（User Interface Designer），指从事对软件的人机交互、操作逻辑、界面美观的整体设计工作的人。UI设计师的涉及范围包括商用平面设计、高级网页设计、移动应用界面设计及部分包装设计，是目前中国信息产业中最为抢手的人才之一。UI设计师进行的是集科学性与艺术性于一身的设计，他们需要完成的，简单说来，正是一个不断为用户设计视觉效果使之满意的过程。
用户研究专员

即将发布，敬请期待

顾名思义，这个职位是研究用户的，属于定性研究范围。用户研究做为一个新兴职业，近几年在IT行业里已经成为了比较成型的流程，已知的设有这一专业职位和团队的公司包括：腾讯、淘宝、金山、网易、苏宁易购、百度、新浪等。不仅如此，做为一个用户研究人员，还需要对于项目背景有比较深刻的了解，比如项目的理念、核心向内容、年度目标等等，这些将帮助用户研究人员来更好地定位需要研究的用户。

技术研发

前端
Web开发工程师
Web前端工程师， 主要职责是利用(X) HTML / CSS / JavaScript/ React /ES6 /jQuery等各种Web前端技术进行产品的研发。 完成客户端程序（ 也就是浏览器端） 的开发， 开发UI界面和JavaScript程序， 同时结合后台开发技术模拟整体效果， 进行丰富互联网的Web开发， 致力于通过技术改善用户体验。 前端技术迅猛发展， 想成为一名合格的前端工程师需要不断的学习以及丰富的实战， 随着HTML5、 CSS3、 Node.js、React、RN等的新技术的不断发展， 前端开发已经向多元化发展，优秀的前端工程师一直是企业高薪追求的对象之一。
H5开发工程师

即将发布，敬请期待

我们通常认为前端是基于浏览器的Web开发，大多运行在PC端，不能在移动端、后端发挥作用，随着HTML5 /CSS3/ES6/Node/RN等技术的逐渐成熟，前端开始像多端发展，技能不在单一的前端工程师有了更高的地位，越来越多人开始转变为全栈工程师。H5技术让用户有了更好的网页浏览体验；Node+js让前端开发能够直接开发后端系统，只需要JS一门语言；React Native 则让更多的人体验到了前端的力量。每一项技术的发展，都需要有相应的人才驾驭它，作为目前最火热的前端发展趋势，这值得你一直学习下去。
微信小程序工程师
微信小程序是一种全新的连接用户与服务的方式，它可以在微信内被便捷地获取和传播，同时具有出色的使用体验，而微信小程序开发是技术人员必备技能之一，它能够综合运用微信提供的生态和前端技术，集设计、前端、后端、运维于一身，为用户提供接近原生的交互体验，是一个新兴的“全栈”发展方向。
移动端
Android工程师
>Android是一种基于Linux的自由及开放源代码的操作系统，由谷歌公司进行维护与开发，主要使用于移动设备(手机、平板、相机、游戏机等)。随着移动互联网时代的到来，作为移动系统的主力军，Android的前景也越来越光明。对于一个公司而言，展示公司的产品，开展公司的业务，移动端都是一个不可或缺的领域。随着智能穿戴设备、智能家居的崛起，移动端也更是大放异彩。因此，Android移动开发前景看好，且就业缺口也十分巨大。对于Android应用的开发可以采用成熟强大的Java语言或者后起之秀--Kotlin语言。
iOS开发工程师
iOS是由苹果公司于2007年研发的一款属于类Unix的封闭式移动操作系统，iOS系统被应用于IPhone、IPad、IPod touch等产品，而基于iOS系统开发的面向对象编程Objective-C语言在这短短十年时间发展迅猛，在移动端APP开发中占有绝对地位。由于开发语言的封闭式，整个iOS生态更加的安全和健壮。随着移动互联网和物联网的不断发展，苹果的市场份额将不断上升，对应的iOS开发将会越来越火爆，从iOS新生的Swift开发语言在短短几年的时间里不断被开发者追捧使用可以看出，iOS开发前景是光明的。牢抓Objective-C这条根，紧抓swift这新宠，iOS开发工程师将会是你走向巅峰的不二之选。
游戏方向
Unity3D工程师

Unity3D是由Unity Technologies开发的一个让玩家轻松创建如三维视频游戏、建筑可视化、实时三维动画等类型互动内容的多平台的综合型游戏开发工具，是一个全面整合的专业游戏3D引擎。U3D游戏引擎凭借操作简单、易学、灵活等优势，已经风靡全球。从Unity官方最新的公布的数据来看 ，Unity游戏引擎已占世界范围内45%的市场份额，居全球首位，而全世界有6亿的玩家在玩使用Unity引擎制作的游戏 ，所以Unity3D游戏开发工程师是全球游戏公司都争抢的高能人才。

后端

  • PHP工程师
    PHP被称为世界上最好的语言，它是一种通用开源脚本语言。语法吸收了C语言、Java和Perl的特点，入门简单，容易掌握，利于学习，使用广泛。PHP语言可操纵多种主流与非主流的数据库、支持面向对象的编程、支持跨平台的操作。PHP无处不在，它可应用于任何地方、任何领域，最广泛的是应用于动态网站开发，比如门户、微博、论坛、电子商务、SNS(社交)等都可以用PHP来实现，除此之外，PHP程序员还可以通过混合式开发APP的方式，将业务领域扩展到移动端（兼容Android和iOS）。从各大招聘网站的数据上来看，PHP开发的职位非常多，薪资水平也很可观，大多数互联网公司都会有对应的岗位，比如新浪、腾讯、阿里、淘宝网、百度、360等等。
  • JavaWeb工程师

    即将发布，敬请期待

    JavaWeb是用Java技术解决相关web互联网领域的技术总和，JavaWeb应用由一组Servlet/JSP、HTML文件、相关Java类，以及其它可以被绑定的资源构成。Java技术对web领域的发展注入了强大的动力。由于Java语言的优势和Java开发人员在编程界的高占比，大部分企业网站都是用JavaWeb开发的，随着互联网不断发展和企业业务的不断扩展，JavaWeb项目开发人员需求正在快速增加，紧随互联网发展形势，成为一名JavaWeb开发工程师将是你不错的选择。
  • Python工程师
    Python 是一门伟大的语言，它于1989年诞生，长期占据TIOBE前10位，被广泛的应用在 Web开发、爬虫开发、数据分析、人工智能、科学运算等各个技术领域，作为一门可“连接”其他语言的胶水语言，他是工程师的最爱的效率工具，因其庞大的类库，让你的项目能够快速的完成试错，随着大数据和AI时代的开启，Python 成为数据专家和AI工程师必备的技能之一，技术的发展日新月异，只有跟上时代的变化，才能成为技术领域的领跑者，学习Python，能让你更快人一步。
  • C/C++工程师

    即将发布，敬请期待

    C++是最贴近底层的编程语言，在性能方面有着无可替代的优势。特别是对于很多游戏开发公司来说，C++尤其适合作为后端服务的开发语言。在一些对于并发性能等要求比较高的业务上，C++也拥有绝对性的优势。因此大部分公司的核心业务都是用C++进行开发。C++开发工程师掌握着公司最核心、重要的业务体系，有着十分重要的地位。
  • Golang工程师
    Go语言是谷歌推出的一种高效编译、高效执行且开发简单的编程语言，Go语言从语言层面支持并发和跨平台编译，因此特别适合做服务器编程、网络编程、分布式系统，数据库代理器等对性能和稳定性要求较高的系统。因此大数据研发工程师、服务器应用研发工程师和云应用服务开发工程师等都是Go语言学习者非常合适的职业方向。另外最近求职市场Go语言工程师需求增长率达到了800%，因此这个职业方向应该是现下最有“钱”途的职业方向了。

大数据

  • 数据研发工程师
    数据研发工程师一般也叫数据开发工程师，基于海量数据的数据仓库建设和数据分析，同时针对各业务场景探索大数据解决方案，主要包括数据处理、数据仓库建模、计算处理、报表开发等。其中一部分工程师是做数据平台的研发的，主要工作是建立高性能的数据ETL系统大规模的处理，存储数据，并且对数据分析提供平台支持。另外一部分工程师是在数据平台基础上开发数据产品的。
  • 数据挖掘工程师

    即将发布，敬请期待

    数据挖掘（英语：Data mining），又译为资料探勘、数据采矿。它是数据库知识发现中的一个步骤。数据挖掘一般是指从大量的数据中通过算法搜索隐藏于其中信息的过程。数据挖掘通常与计算机科学有关，并通过统计、在线分析处理、情报检索、机器学习、专家系统（依靠过去的经验法则）和模式识别等诸多方法来实现上述目标。
  • 数据分析工程师

    即将发布，敬请期待

    数据分析工程师是基于业务的，主要负责业务数据收集整理，搭建业务数据体系，结合业务对多种数据源进行深度诊断性组合分析、挖掘、深度分析，通过从复杂的数据里面发现和分析业务问题，提炼核心结果，及时发现和分析其中隐含的变化和问题，为决策提供信息支持，比较强调的能力是公司所在行业的领域知识，数据分析能力，对数据的敏感程度，数据可视化技术，沟通能力，特别是面向非技术职能人员的共同能力。

AI

  • 机器学习工程师

    即将发布，敬请期待

    机器学习(Machine Learning, ML)是人工智能的分支，是一门多领域交叉学科，涉及概率论、统计学、逼近论、凸分析、算法复杂度理论等多门学科。专门研究计算机怎样模拟或实现人类的学习行为，以获取新的知识或技能，重新组织已有的知识结构使之不断改善自身的性能。它是人工智能的核心，是使计算机具有智能的根本途径，其应用遍及人工智能的各个领域。
  • 深度学习工程师

    即将发布，敬请期待

    众所周知，从去年的AlphaGo到今年人工智能首次写进政府工作报告，人工智能正在席卷全球，而AI的核心技术是机器学习和深度学习。去年3月AlphaGo大胜李世石到今年5月再胜世界排名第一的柯洁，更是奠定了AI的领军地位，毫无疑问的代表了科技最前沿、最潮流的技术。
  • AI算法工程师

    即将发布，敬请期待

    近年来，人工智能技术在各行各业中的应用越来越普及，相关专业技术人才也是供不应求，各大公司或是创业公司不惜重金招募AI人才。最近一项统计显示，人工智能相关职位平均年薪达到30万元-60万元，从业时间长的甚至能达到年薪百万。优秀的算法是推动技术进步的必备属性，随着人工智能在各领域的普及深入，AI算法工程师更是各大公司争抢的对象。

支撑

  • 运维工程师

    即将发布，敬请期待

    运维工程师主要负责公司服务器的稳定性，确保整个系统的高可用性。在此基础上对服务架构进行不断的优化，提升部署效率、优化资源利用率提高整体的ROI.因此，运维工程师是保证公司服务系统能够正常高效运行的一个不可或缺的位置。随着互联网时代的发展，科技公司越来越多，而运维工程师的缺口预计会达到40w左右，掌握运维技术将会扩宽你的职业道路。
  • 数据库工程师

    即将发布，敬请期待

    数据库开发工程师(Database Developer)是从事数据库管理系统(DBMS)和数据库应用软件设计研发的相关工作人员的统称，他属于软件研发工程师，但又有一部分运维工作的内容。他主要从事软件研发的工作，但同时也要参与数据库生产环境的问题优化和解决。与普通的数据库管理员(DBA)不同的是，他是属于研发的职位，在基本的数据库运维基础上，还需要参与一些产品数据架构的设计开发等工作
  • 安全工程师

    即将发布，敬请期待

    在这个信息通畅发达的时代，安全成了各大公司极其重视的一个职业。安全工程师，不仅需要具备基础的编程能力。更需要懂得底层网络协议及各种数据加密传输的门道，甚至是系统内核破解等高深的技术。是目前市场上供不应求的职业之一。

QA工程师

即将发布，敬请期待

QA即英文QUALITY ASSURANCE 的简称，中文意思是品质保证，其在ISO8402：1994中的定义是“为了提供足够的信任表明实体能够满足品质要求，而在品质管理体系中实施并根据需要进行证实的全部有计划和有系统的活动”。有些推行ISO9000的组织会设置这样的部门或岗位，负责ISO9000标准所要求的有关品质保证的职能，担任这类工作的人员就叫做QA人员。

如何开始学习

了解职业

首先，你需要了解职业的定位和互相的关系，知晓每个职业的特点。

选择适合你的职业

然后，基于个人特点，选择适合自己，同时也是自己喜欢的职业方向。

构建个性学习路径

接着，通过技能测评，找出缺失的技能模块，构建个性化的学习路径。

持续学习和实战

在导师/教练的指导和同学帮助下，不断的学习和训练，在做中学，实战提高。

目标达成

学习完成后，提升自己技能，获得更理想工作，开启美好前程。

学习过程

  • 1 - 课程学习

    在极客学院构建的学习路径，通过立体化的课程（视频，图文，考题等），先进行理论和知识的学习，构建自己的知识体系和知识图谱，为后续动手实战积累知识；在学习过程中，如果遇到问题，还会得到老师的指导和同学的帮忙，让学习更有效。

  • 2 - 项目实战

    除了知识和理论，职业中最重要的是需要实际解决问题的能力，通过极客学院构建的实战项目，从易到难，从小到大，从简单到复杂，从虚拟到真实。在这个过程中，你需要真正动手做很多的项目，在做的过程中完善理论，学会分析问题和解决问题，并真正学以致用到以后的实际工作中。

  • 3 - 结果评定

    没有反馈的实战是枯燥的，在极客学院的学习系统中，一线的专家工程师会对你的学习结果和实战项目予以评判和指导，你将会进一步完善项目，精益求精。此外，你也可以参与到其他同学的项目评定中，汲取所长，进一步完善自己的项目。
